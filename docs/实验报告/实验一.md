# 实验一：数据准备与预处理

## 实验目的

- 熟悉数据预处理流程
- 掌握Hive外部表的创建和使用
- 了解大数据处理的基本步骤

## 实验步骤

### 1. 数据准备

1. **检查数据集**：确保 `/export/software/data_format.zip` 存在且可解压
2. **解压数据**：解压后应有 `user_log.csv`、`train.csv`、`test.csv`
3. **数据预处理**：删除 `user_log.csv` 首行，生成 `predeal.sh` 脚本
4. **生成小数据集**：生成 `small_user_log.csv`，确保正好 100000 行，只含 `month=11`、`day=11` 的数据

### 2. 数据上传

1. **创建HDFS目录**：`hdfs dfs -mkdir -p /user/hive/warehouse/dbtaobao/dataset/user_log`
2. **上传数据**：`hdfs dfs -put small_user_log.csv /user/hive/warehouse/dbtaobao/dataset/user_log`

### 3. Hive建表

1. **创建数据库**：`hive -e "create database if not exists dbtaobao;"`
2. **创建外部表**：
   ```sql
   create external table dbtaobao.user_log(
       user_id string,
       item_id string,
       cat_id string,
       merchant_id string,
       brand_id string,
       month int,
       day int,
       action int,
       age_range int,
       gender int
   )
   row format delimited fields terminated by ','
   location '/user/hive/warehouse/dbtaobao/dataset/user_log';
   ```

### 4. 数据验证

1. **验证数据行数**：`hive -e "select count(*) from dbtaobao.user_log;"`
2. **验证数据质量**：`hive -e "select * from dbtaobao.user_log limit 10;"`

## 实验结果

- **数据行数**：100000
- **数据质量**：正常，无乱码
- **Hive表状态**：外部表创建成功，可正常查询

## 实验结论

实验一数据准备与预处理完成，数据已成功上传到HDFS并创建了Hive外部表，为后续实验做好了准备。
