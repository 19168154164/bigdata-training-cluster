# 实验五：数据可视化

## 实验目的

- 熟悉Web应用的部署和配置
- 掌握ECharts数据可视化的基本使用方法
- 了解大数据可视化的常见场景

## 实验步骤

### 1. 准备工作

1. **检查Tomcat**：确保Tomcat安装在`/export/servers/tomcat-8.5.88`
2. **准备JDBC驱动**：确保MySQL JDBC驱动在Tomcat的lib目录中

### 2. 部署Web应用

1. **创建MyWebApp**：
   - 创建目录结构：`/export/servers/tomcat-8.5.88/webapps/MyWebApp`
   - 创建`WEB-INF`、`css`、`js`等目录

2. **创建JSP页面**：
   - `index.jsp`：消费行为统计
   - `index2.jsp`：男女交易对比
   - `index3.jsp`：年龄分布对比
   - `index4.jsp`：商品类别销量
   - `index5.jsp`：省份成交量
   - `index6.jsp`：回头客预测

3. **创建CSS文件**：`css/style.css`，实现现代化的UI设计

4. **创建Java类**：`WEB-INF/classes/dbtaobao/connDb.java`，实现数据库连接

### 3. 配置和启动

1. **编译Java类**：
   ```bash
   javac -cp /export/servers/tomcat-8.5.88/lib/servlet-api.jar:/export/servers/tomcat-8.5.88/lib/mysql-connector-java-8.0.28.jar -d /export/servers/tomcat-8.5.88/webapps/MyWebApp/WEB-INF/classes /export/servers/tomcat-8.5.88/webapps/MyWebApp/src/dbtaobao/connDb.java
   ```

2. **启动Tomcat**：
   ```bash
   cd /export/servers/tomcat-8.5.88
   ./bin/startup.sh
   ```

3. **验证Tomcat启动**：
   ```bash
   ps aux | grep tomcat | grep -v grep
   ```

### 4. 验证网页访问

1. **访问主页面**：`http://192.168.121.138:8080/MyWebApp/index.jsp`
2. **访问各功能页面**：
   - 男女交易对比：`http://192.168.121.138:8080/MyWebApp/index2.jsp`
   - 年龄分布对比：`http://192.168.121.138:8080/MyWebApp/index3.jsp`
   - 商品类别销量：`http://192.168.121.138:8080/MyWebApp/index4.jsp`
   - 省份成交量：`http://192.168.121.138:8080/MyWebApp/index5.jsp`
   - 回头客预测：`http://192.168.121.138:8080/MyWebApp/index6.jsp`

3. **验证页面功能**：
   - 无404错误
   - 无500错误
   - 图表正常渲染
   - 数据正确显示

## 实验结果

- **图表显示**：所有图表正常渲染
- **数据显示**：数据正确显示，无乱码
